# Face Recognition System using ArcFace (DeepFace)

This project implements a **face recognition system** using **ArcFace embeddings**
extracted via **DeepFace**, with GPU acceleration (NVIDIA RTX).

---

## 🚀 Features
- ArcFace deep embeddings
- RetinaFace detection
- GPU acceleration (TensorFlow + CUDA)
- Cosine similarity–based recognition
- No classifier (scales to unknown faces)
